# Gmail Draft Workflow

Gmail은 실제 발송이 아니라 초안 생성까지만 사용한다.

## Inputs

- 수신자 이름
- 수신자 이메일
- 매장명
- Netlify 리포트 URL
- 핵심 지표 1~2개
- 담당자명

## Draft Rules

- 실제 발송은 하지 않는다.
- 초안 생성 전 수신자 이메일과 본문을 사용자에게 보여준다.
- 링크가 올바른 Netlify URL인지 확인한다.
- 고객 정보는 mock 또는 공개 가능한 샘플만 사용한다.

## Subject Template

```text
[매장명] 티오더 주문 리포트 공유드립니다
```

## Body Template

```text
안녕하세요, [수신자명] 사장님.

[기간] 티오더 주문 데이터를 바탕으로 매장 리포트를 정리해 공유드립니다.
총 주문 금액은 [총액]원, 추가 주문율은 [추가주문율]%로 확인되었습니다.

아래 링크에서 모바일로 바로 확인하실 수 있습니다.
[Netlify URL]

확인 후 궁금하신 점이 있으면 편하게 말씀 부탁드립니다.

감사합니다.
[담당자명] 드림
```

## Gmail MCP Behavior

Gmail 도구가 연결되어 있으면 draft를 만든다. 연결되어 있지 않으면 위 템플릿을 사용해 초안 텍스트만 제공한다.
